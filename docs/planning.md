## 앱 개요

장보기 기록을 관리하고, 영수증 촬영으로 품목을 자동 등록하는 개인용 쇼핑 기록 앱.

### 배포 전략

- 1단계: 개인용으로 개발 및 사용
- 2단계: 사용감 검증 후 광고 또는 인앱 결제를 붙여 퍼블릭 배포

---

## 핵심 기능

### 1. 장보기 기록

- 구매한 품목과 가격을 기록
- 품목별 / 가격별로 정리하여 조회 가능
- 다음 장볼 때 이전 기록을 참조해 빠르게 목록 구성

### 2. 영수증 자동 인식

- 영수증 사진 촬영 시 품목 자동 파싱 및 등록
- **Google ML Kit** (Text Recognition) 사용 예정
- 인식된 품목은 수동 수정 가능
